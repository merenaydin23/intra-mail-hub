# Intra Mail Hub - Corporate Internal Email System

## 📌 Project Overview
[cite_start]Intra Mail Hub is a secure and structured internal communication platform designed to address the complex hierarchical needs of modern corporate environments[cite: 5, 24]. [cite_start]Developed as a core Software Engineering project, it focuses on architecting robust data flow patterns and maintaining high-integrity communication protocols[cite: 24, 25, 28].

## 🚀 Key Features
* **Fabrika & Bayi Hiyerarşisi:** 4 ana kategoride (Fabrika, Bölge Bayi, Yerel Bayi, Yerel Bayi Çalışanı) özelleştirilmiş yetkilendirme sistemi.
* **Merkezi Yönetim (Admin):** Fabrika admini için tüm sistem trafiğini isim bazlı denetleme ve izleme paneli.
* **Yapay Zeka Destekli:** Gemini API ile mesaj özetleme, resmileştirme (formalize) ve akıllı yanıt önerileri.
* **Güvenli İletişim:** `@intramail.corp` uzantılı hesaplar arası güvenli veri akışı.

## 📂 Repository Structure
* `giris.html`: Sisteme ana giriş (Login) ekranı.
* `iletisim.html`: Bayi ve personel için ana mesajlaşma portalı.
* `yonetim.html`: Fabrika genel yöneticisi için kontrol paneli.
* `/src`: Mesajlaşma ve yönetim çekirdek kodları.
* `/docs`: Sistem Gereksinimleri Spesifikasyonu (SRS) ve teknik dökümantasyon.
